# Week 01 — Linux Fundamentals & CLI Navigation

**Date:** January 2026
**Program:** Knowledge House Cybersecurity Fellowship

## Summary

This session introduced the Linux operating system with a focus on
command-line interface navigation and foundational system administration
skills. Students explored the Linux file system hierarchy and practiced
essential commands used in cybersecurity operations (Knowledge House, 2026).

## Tools & Commands Used

- `pwd` — print working directory
- `ls`, `ls -la` — list directory contents
- `cd` — change directory
- `mkdir`, `rm`, `cp`, `mv` — file management
- `cat`, `less`, `grep` — file reading and searching
- `chmod`, `chown` — file permissions
- `man` — manual pages for command reference

## Key Takeaways

Understanding the Linux file system hierarchy is foundational to all
cybersecurity work, as most servers and penetration testing tools run on
Linux-based systems. Mastery of the command line enables faster and more
precise interaction with target systems during assessments. File permission
management using `chmod` and `chown` is critical for both exploiting
misconfigurations and hardening systems against attack.
