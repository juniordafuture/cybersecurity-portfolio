# Week 02 — Networking Basics & Protocols

**Date:** January 2026
**Program:** Knowledge House Cybersecurity Fellowship

## Summary

This session covered foundational networking concepts including the OSI
model, TCP/IP protocol suite, IP addressing, and subnetting. Understanding
how data travels across networks is essential for identifying attack surfaces
during penetration tests (Forouzan, 2021).

## Tools & Commands Used

- `ip a` — view network interfaces and IP addresses
- `ping` — test host reachability
- `traceroute` — trace packet path to destination
- `netstat` — display network connections
- `ss` — socket statistics
- `ifconfig` — legacy network interface configuration
- `whois` — domain registration lookup

## Key Takeaways

The OSI model provides a structured framework for understanding where
vulnerabilities exist within a network stack, from the physical layer up
to the application layer. TCP/IP is the foundation of internet
communication, and understanding port numbers and protocols such as HTTP,
FTP, SSH, and DNS is critical for network reconnaissance. Subnetting skills
allow security professionals to map and segment networks effectively during
an engagement.
