4. REMEDIATION
* What is the primary defense against this attack?
Answer: The primary defense is Parameterized Queries (also called 
Prepared Statements). Instead of building SQL strings with user input 
directly, parameterized queries separate the SQL code from the data, 
making it impossible for injected input to alter the query logic. 
This prevents attackers from manipulating the database regardless 
of what they type into input fields.
